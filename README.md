# Mall of America Interactive Sales Deck

## Overview
Fully interactive browser-based sales tool for Mall of America, world's largest US shopping destination. Video-first, luxury UI, non-linear navigation. Targets tenants, sponsors, event partners.

**Live Demo:** https://sriharikunchala18.github.io/moa-sales-deck

**Tech Stack:**
- HTML5, CSS3 (Flex/Grid, animations)
- Vanilla JavaScript (IntersectionObserver, smooth scroll)
- Public assets from mallofamerica.com & YouTube
- Responsive (desktop/tablet), performant (lazy load)

## Setup
1. Clone repo
2. Open `index.html` in browser or use `npx serve .` / Python HTTP server
3. Deploy: GitHub Pages (gh-pages branch) or Netlify drop
